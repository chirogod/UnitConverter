Aplicacion web sencilla que permite convertir entre diferentes unidades de medida. Puede convertir unidades de longitud, peso, temperatura.
URL de la pagina del proyecto: https://roadmap.sh/projects/unit-converter

# Requisitos

- El usuario ingres el valor a convertir.
- Selecciona la unidad de origen.
- Selecciona el destino de la conversion.
- Se visualiza el resultado de la conversion.

# Instalacion
Para usar la aplicacion, seguir los siuientes pasos:
1. Clonar el repositorio:

       git clone https://github.com/chirogod/UnitConverter.git 

2. Navegar a la raiz del proyecto.

		cd UnitConverter

3. Correr la aplicacion.

		dotnet run

4. Pegar en el navegador la direccion dada en consola, ejemplo:

		Now listening on: http://localhost:5010
